# 📝 面接スクリプト / 日文面試稿

日文面試準備用靜態網站，支援快速切換公司與跳轉至對應問題區塊。

🔗 **網站連結**：[開啟面試稿](https://hcjohn463.github.io/interviewJP/)

---

## ✨ 功能

- 🏢 **雙公司切換**：IT企業 / 鋼鐵企業（Tab 頁籤切換）
- 📌 **快速跳轉**：點擊按鈕直接跳到指定問題區塊，並亮起黃色邊框標記
- ↑ **回到頂部**：右下角浮動按鈕，快速回到按鈕導覽區
- 🌙 **深色模式**：護眼設計，適合面試前快速瀏覽
- 📱 **手機友善**：RWD 響應式排版

## 📂 檔案結構

```
interviewJP/
├── index.html   # 主頁面（稿子內容在這裡）
├── style.css    # 深色主題樣式
├── script.js    # Tab 切換 / 跳轉 / 回頂邏輯
└── README.md
```

## 🛠️ 部署

本站部署於 [GitHub Pages](https://pages.github.com/)，無需後端，純靜態網站。

更新稿子後執行：

```bash
git add .
git commit -m "update scripts"
git push
```
